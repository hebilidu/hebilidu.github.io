CSS and styling

I have heard quite a few people being contemptuous upon CSS. The main argument being that lots of tools generate beautiful styles automatically. It is a bit like despising a car engine because it is hidden under the hood. Who developped those tools ? CSS experts ! Mastering CSS is quite an intellectual challenge!
